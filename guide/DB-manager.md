# 数据库管理
SQL Server 数据库管理为用户提供数据库级别的基本运维能力，包括数据库的创建、删除与查看。

## 查看数据库
**前置条件**

已成功创建实例，当前处于运行中状态

**操作步骤**

1. 登录控制台  [**UDB SQLServer 实例列表**](https://console.ucloud.cn/udb/sqlserver)，点击目标实例的操作列“**详情**”按钮进入详情页；
2. 选择“**数据库管理**”标签页，查看已经创建的数据库列表。

![](/images/1764062631497-0b28aef3-5ec0-48fc-911c-c44f48c2860f.png)

## 创建数据库
若要使用云数据库 SQLServer 版，您需要在实例中创建数据库。本文介绍如何为 SQLServer 实例创建数据库。

**前置条件**

已成功创建实例，当前处于运行中状态。

**操作步骤**

1. 登录控制台  [**UDB SQLServer 实例列表**](https://console.ucloud.cn/udb/sqlserver)，点击目标实例的操作列“**详情**”按钮进入详情页；
2. 选择“**数据库管理**”标签页，点击“**创建数据库**”按钮；
3. 在弹窗中输入数据库名称、授权账号，点击 **“确定”** 保存配置。

> 说明：授权账号下拉列表仅展示通过控制台创建的授权账号，如需新增授权账号，请参考 [账号管理](/udb-sqlserver/quick/account-manager) 相关文档。
>

## 删除数据库
**前置条件**

+ 已成功创建实例，当前处于运行中状态
+ 已为实例创建了数据库

**操作步骤**

1. 登录控制台  [**UDB SQLServer 实例列表**](https://console.ucloud.cn/udb/sqlserver)，点击目标实例的操作列“**详情**”按钮进入详情页；
2. 选择“**数据库管理**”标签页，找到目标数据库，点击“**删除**”按钮，并“确认”删除。

![](/images/1764063231557-4c7d8c58-9c05-4c89-8f28-34c5e487a48d.png)
